# [WIP] Jayelix's `dotfiles`

Repository which bootstraps every of my new `linux`/`darwin` system instance to make my CLI fancy and productive.

## Reqiurements
Before everything you should set `zsh` as a login shell.

```
chsh -s $(which zsh)
```
